# Instituto Provincial de Educación Terciaria N° 1

Este sitio web presenta información institucional, carreras, contacto y novedades del IPET N°1. Fue desarrollado como práctica de estructura semántica en HTML, uso de CSS externo, diseño responsivo con Flexbox y Grid, e integración con Formspree para el formulario de contacto.

## Tecnologías Usadas

- HTML5
- CSS3
- Flexbox y Grid
- Google Fonts
- Formspree
